privacy

By choosing to use this app, you agree to allowing the use of your Spotify playlist and library data, and the ability to modify the playback state of your Spotify. In other words, this app will create a new playlist based on your preexisting library, and if you choose to use the random shuffler tool, it will add a specified number of random songs to the queue.

None of the data used by this app is stored, collected, or shared anywhere. This application uses the authorization code flow with PKCE.

If you would like to revoke permissions for the All Songs Playlist app, please visit your apps page and click "REMOVE ACCESS" for the All Songs Playlist application. See the about page to visit my GitHub and inspect the code.
